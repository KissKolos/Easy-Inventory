
\section{\mt{Web frontend}{}}

\subsection{\mt{Áttekintés}{}}

\p{\mt{%
	A webes frontend tesztelésére csupán E2E autómata teszteket használunk, mivel ezek kellően letesztelik azt, így nincs szükség további tesztekre.
}{}}

\subsection{\mt{E2E tesztek}{}}

\p{\mt{%
	Az E2E tesztekhez python alapú playwright keretrendszert használunk.
	Az E2E tesztek minden műveletet végrehajtanak kattintás és gépelés szimulációval.
	Helyes és hibás bemeneti adatok egyaránt tesztelve vannak, illetve a tesztek egy része képernyő képeket készít a program működéséről, amiket a dokumentációhoz használunk fel.
}{}}
