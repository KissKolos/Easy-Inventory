\section{\mt{Mobil alkalmazás}{}}

\block{%
	\subsection{\mt{Áttekintés}{}}

	\p{\mt{%
		A mobil alkalmazás tesztelése J4Unit teszt keretrendszerrel történik.
	}{}}
}

\block{%
	\subsection{\mt{E2E tesztek}{}}

	\p{\mt{%
		A mobil alkalmazás tesztelése olyan felhasználói felület tesztekkel történik, amellyek kattintásokat és szövegek beírását szimulálják.
		Ezek a tesztek ugyanazt tesztelik, amit az asztali alkalmazás E2E tesztjei, annyi külömbséggel, hogy ezek az \code{Espresso} UI tesztelési keretrendszert használják a felhasználói interakciók szimulálására.
		A tesztekkel helyes és hibás bemeneti adatok is egyaránt tesztelve vannak, illetve a tesztek egy része képernyőképeket készít, amelyeket a dokumentációhoz használunk fel.
	}{}}
}
